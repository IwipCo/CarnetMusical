Configuration GitHub

Pour sauvegarder directement tes ajouts sur le catalogue distant, entre ici un Personal Access Token (PAT) GitHub ayant les droits d'écriture sur le dépôt IwipCo/CarnetMusical.
Ce token reste 100% sécurisé en local dans ton navigateur.
Génère un lien contenant ce token : la personne qui l'ouvre sera connectée automatiquement, sans avoir à le coller elle-même. ⚠️ Quiconque a ce lien peut écrire sur le dépôt — ne le partage qu'à des personnes de confiance, et ne le colle jamais dans un espace public.
